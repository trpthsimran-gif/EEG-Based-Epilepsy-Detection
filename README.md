# EEG-Based Epilepsy Detection using ML & DL

##  Overview
This project focuses on detecting epilepsy using EEG signals with Machine Learning and Deep Learning techniques.

##  Key Features
- Preprocessed EEG data (6,000+ epochs)
- Feature engineering (PLV, Hjorth parameters)
- ML models: Random Forest, SVM, XGBoost
- DL model for classification
- Achieved AUC up to 0.94

## 🛠 Tech Stack
- Python, NumPy, Pandas
- Scikit-learn
- TensorFlow / Keras
- Matplotlib, Seaborn

##  Results
- Improved model performance from AUC 0.56 → 0.94
- Reduced noise and enhanced feature quality

##  Files
- `Pure_ML_Pipeline.ipynb` – Classical ML pipeline
- `DL_Classification.ipynb` – Deep Learning model
- `Clinical_Biomarker_EEG.ipynb` – Feature extraction
